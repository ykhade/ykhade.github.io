Work

I spend most of my working hours writing code for people…

TODO-App | A todo app

A iOS and Android app written in the hybrid Flutter framework to create a clean app. Tools used include Flutter and Dart.

ArtiFilter | Neural Art Generator

ArtiFilter is a web app that uses deep learning to convert ordinary photos into artwork inspired by Matisse, Picasso, Van Gogh, and other great artists. Tools used include Angular, TypeScript, Flask, and GCP.

FlashLawyer | ChatBot Powered Legal Documents

FlashLayer provides a quick, free solution for commonly used legal documents. Unlike most apps that require a complex signup process, FlashLawyer makes the document available in real-time with chatbot assistance. Tools used include Angular, AWS Lex, and Firebase

CampusHippo | Short-Burst Learning

CampusHippo is a short-burst learning platform that allows people to share wisdom about tech, business, and marketing. Rather than teach a lesson about a single broad topic, CampusHippo targets highly specialized components of various skills. Tools used in Ruby on Rails, AngularJS, Redis, and Stripe Payments.

HippIcons | An Alternative Icon Font

HippIcons is an open source icon font designed as a drop-in alternative to FontAwesome. It matches the class names of FontAwesome, but uses a completely different icon design, allowing developers to painlessly change icon styles in a web application. Tools used include MeteorJS, Sass, Adobe Illustrator, and AWS CloudFront.

Anchorage.io | Community-Powered Boating Guides

Anchorage.io is a community-driven open-source project that enables sailors and other boat people to share knowledge and photos of remote boating locations. When you sail offshore to an unexplored anchorage, reliable information is scarce. Anchorage.io empowers boaters to share their opinions and concerns about any location on the water. Tools used in Ruby on Rails, Forecast.io, and Segment Analytics.

CodeDiode.io | Code Tutorials Off the Beaten Path

CodeDiode.io is a side project that focuses on step-by-step tutorials about writing code. Whenever I have time to share some insight with the web development community, I post it here.
